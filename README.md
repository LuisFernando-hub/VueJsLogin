
# Project Task Registration System

Project with Login and Task Registration system

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles the backend
```
npm run backend
```


### Compiles and minifies for production
```
npm run build
```
